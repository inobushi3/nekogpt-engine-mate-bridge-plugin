# Bridge protocol

NekoGPT runs a local WebSocket server on `127.0.0.1`.

Default port:

```text
17878
```

The Engine Mate plugin connects as a client and must send the pairing token generated by NekoGPT.

## Events sent by NekoGPT

- `pair`: validates the local token.
- `setExpression`: changes expression.
- `speechStart`: starts subtitle and lip sync.
- `speechEnd`: stops subtitle and lip sync.
- `mouth`: controls mouth openness.
- `immersiveTool`: pizza, lollipop, hammer and perfume.
- `music`: dance start/stop events.

## Events sent by Engine Mate

- `status`: connection and runtime state.
- `hitbox`: avatar interactions.
- `patpat`: head hover/pat event.
- `boundaryTouch`: sensitive/body boundary interaction.
- `error`: clear runtime error.

All messages are JSON and stay on localhost.
